> Visual Studio에서 코딩 생산성을 높여주는 기본 조작, 코드 편집, 리팩터링 및 디버깅 단축키 모음입니다.

💡 **꿀팁:** Visual Studio에서 `Ctrl + Q`를 누르면 “모든 명령 검색”이 열립니다.

## 기본 조작
- `Ctrl + Shift + N`: 새 프로젝트 만들기
- `Ctrl + Shift + A`: 새 항목 추가
- `Ctrl + O`: 파일 열기
- `Ctrl + S`: 저장
- `Ctrl + Shift + S`: 모두 저장
- `Alt + T + O`: 옵션 바로가기
- `Ctrl + Tab`: 열려 있는 파일(탭) 전환
- `←` (왼쪽 화살표): 선택한 노드 접기 (Collapse)
- `→` (오른쪽 화살표): 선택한 노드 펼치기 (Expand)
- `Ctrl + M, Ctrl + M`: 현재 블록 토글
- `Ctrl + M, Ctrl + O`: 전체 접기
- `Ctrl + M, Ctrl + L`: 전체 펼치기
- `Ctrl + Alt + L`: 솔루션 탐색기 열기

## 코드 편집
- `Ctrl + K, Ctrl + C`: 선택한 줄 주석 처리
- `Ctrl + K, Ctrl + U`: 선택한 줄 주석 해제
- `Ctrl + D`: 현재 줄 복사 (Duplicate)
- `Ctrl + X`: 잘라내기
- `Ctrl + Z`: 실행 취소
- `Ctrl + Y`: 다시 실행
- `Shift + Enter`: Smart Enter (자동으로 `;` 찍고 엔터)

## 코드 자동 생성 (Snippet)
- `svm` + `Tab`: `static void Main()` 메소드 생성
- `cw` + `Tab`: `Console.WriteLine()` 메소드 생성

## 탐색 및 찾기
- `Ctrl + F`: 찾기
- `Ctrl + H`: 바꾸기
- `F12`: 정의로 이동
- `Alt + F12`: 미리 보기로 정의 보기 (Peek Definition)
- `Ctrl + -`: 이전 위치로 이동
- `Ctrl + Shift + -`: 다음 위치로 이동

## 코드 실행 & 디버깅
- `F5`: 디버깅 시작
- `Ctrl + F5`: 디버깅 없이 실행
- `F9`: 중단점 설정/해제
- `F10`: 한 단계씩 실행 (Step Over)
- `F11`: 한 단계씩 코드 내부로 이동 (Step Into)
- `Shift + F11`: 현재 함수 나가기 (Step Out)

## 코드 리팩터링 / 자동화
- `Ctrl + .`: 빠른 수정 / 제안 보기 (Quick Actions)
- `Ctrl + K, Ctrl + D`: 코드 자동 정렬
- `Ctrl + R, Ctrl + R`: 변수명 한꺼번에 변경 (Rename)
