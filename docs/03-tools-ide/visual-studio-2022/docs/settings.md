> Visual Studio 2022의 유용한 설정(CodeLens 참조 표시 제거 등)을 정리합니다.

## CodeLens 참조 표시 제거

에디터 상단에 표시되는 CodeLens(참조 횟수 등)가 가독성을 해칠 경우 아래 경로에서 해제할 수 있습니다.

- **도구** → **옵션** → **텍스트 편집기** → **모든 언어** → **CodeLens** → `CodeLens 사용` 체크 해제
