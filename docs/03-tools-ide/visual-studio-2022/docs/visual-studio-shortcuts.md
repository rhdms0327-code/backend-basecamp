> 개발 생산성을 높여주는 필수 단축키들을 정리했습니다.

## 1. 기본 조작
- **Ctrl + Q**: 모든 명령 검색 (핵심!)
- **Ctrl + Shift + N**: 새 프로젝트 만들기
- **Ctrl + Tab**: 열려 있는 파일(탭) 전환
- **Ctrl + S / Ctrl + Shift + S**: 현재 파일 저장 / 모두 저장

## 2. 코드 편집
- **Ctrl + K, C / Ctrl + K, U**: 선택 영역 주석 처리 / 주석 해제
- **Ctrl + K, D**: 코드 자동 서식 정리 (코드 정렬)
- **F12**: 정의로 이동 (Go to Definition)
- **Ctrl + . (마침표)**: 빠른 수정 및 리팩토링 제안 보기

## 3. 빌드 및 디버깅
- **Ctrl + Shift + B**: 솔루션 빌드
- **F5**: 디버깅 시작
- **Ctrl + F5**: 디버깅 없이 시작
- **F9**: 중단점(Breakpoint) 설정/해제
- **F10 / F11**: 프로시저 단위 실행 / 한 단계씩 코드 실행

## 4. 창 관리
- **Ctrl + Alt + L**: 솔루션 탐색기 열기
- **Ctrl + \ , E**: 에러 목록 보기

---
> **팁**: `Tools > Options > Environment > Keyboard`에서 모든 단축키를 확인하고 커스터마이징할 수 있습니다.
