> VS Code에서 .NET 개발 및 일반적인 코딩 작업을 수행할 때 생산성을 높여주는 주요 단축키를 표로 나누어 정리합니다.

---

## 1. 필수 토글(Toggle) 단축키 모음

개발 중 수시로 껐다 켰다(Toggle) 해야 하는 필수 기능들입니다.

| 기능 | 단축키 |
| :--- | :--- |
| **터미널 토글** | ``Ctrl + \`` |
| **사이드바 토글** | `Ctrl + B` |

## 2. C# 및 코드 편집 필수 단축키

| 기능 | 단축키 |
| :--- | :--- |
| **Import 정리 및 정렬** | `Shift + Alt + O` |
| **이름 일괄 변경** | `F2` |
| **정의로 이동** | `F12` |
| **빠른 수정 제안** | `Ctrl + .` |

## 3. 탐색 및 창 이동 단축키

| 기능 | 단축키 |
| :--- | :--- |
| **파일 빠른 검색** | `Ctrl + P` |

## 4. 코드 접기(Folding) 및 문서 이동 단축키

| 기능 | 단축키 |
| :--- | :--- |
| **전체 코드 접기** | `Ctrl + K` -> `Ctrl + 0` |
| **전체 코드 펴기** | `Ctrl + K` -> `Ctrl + J` |
