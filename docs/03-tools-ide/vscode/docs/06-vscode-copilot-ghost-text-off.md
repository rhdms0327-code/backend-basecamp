> 코딩 중 시야를 방해하는 Github Copilot의 흐릿한 회색 자동완성(Ghost Text) 제안을 끄는 방법입니다.

코딩하다 보면 코파일럿이 흐릿한 회색 글씨로 코드를 미리 보여줄 때가 있죠? 이걸 **Ghost Text(유령 텍스트)**라고 부릅니다. 공부할 때는 이 유령들이 내 생각을 방해하지 않도록 잠시 숨겨두는 것이 좋습니다.

## ✅ 설정 방법 (정밀 가이드)

1. **아이콘 클릭**: VS Code 우측 하단 상태바에서 귀여운 Copilot 아이콘(🤖)을 클릭합니다.
2. **메뉴 진입**: 팝업 메뉴에서 **Inline Suggestions** 항목을 찾습니다.
3. **체크 해제**: `Ghost Text suggestions` 옆의 체크박스를 눌러 해제합니다.
