> VS Code에서 마우스 휠을 사용하여 편리하게 글자 크기를 조절하는 방법입니다.

---

## 설정 방법

1. **설정창 열기**: `Ctrl + ,` (콤마)를 누릅니다.
2. **검색**: 상단 검색창에 `mouse wheel zoom`이라고 입력합니다.
3. **체크**: `Editor: Mouse Wheel Zoom` 항목의 체크박스를 체크합니다.

## 결과

이제 코드 창에서 **Ctrl을 누른 채 휠을 돌려보세요**. 아주 부드럽게 글자 크기가 커지고 작아질 겁니다.
