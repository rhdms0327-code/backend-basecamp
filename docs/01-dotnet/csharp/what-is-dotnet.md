# [.NET 분석] .NET이란 무엇인가?

## 1. 플랫폼(Platform)으로서의 .NET
.NET은 클라우드, 웹, 데스크톱, 모바일, 게임 등 다양한 애플리케이션을 개발, 빌드, 실행할 수 있는 **전체 환경**을 의미합니다.

## 2. 개발 가능한 앱 종류
- **웹/클라우드**: ASP.NET Core
- **데스크톱**: WPF, WinForms
- **모바일**: MAUI
- **게임**: Unity (C# 기반)
- **콘솔**: CLI 기반 도구

## 3. 다국어 지원
.NET의 실행 엔진인 **CLR(Common Language Runtime)**은 여러 언어(C#, F#, VB.NET 등)로 작성된 코드를 **IL(중간 언어)**로 통일하여 실행할 수 있게 해줍니다.

## 4. .NET의 구성 요소
- **런타임 (Runtime)**: 프로그램이 실행되는 엔진.
- **BCL (Base Class Library)**: 문자열 처리, I/O 등 공통으로 쓰이는 표준 라이브러리.
- **SDK**: 개발에 필요한 컴파일러와 도구 모음.
