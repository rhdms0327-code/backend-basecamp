# [1장] C# 파일 구조 및 환경 설정

## 1. 주요 용어 및 확장자
- **.cs**: C# 소스 코드 파일.
- **.csproj**: C# 프로젝트 파일. 프로젝트 설정 및 참조 정보를 담고 있습니다.
- **.sln**: 솔루션 파일. 여러 개의 프로젝트를 하나로 묶어 관리합니다.

## 2. 컴파일 과정
1. **소스 코드**: 개발자가 작성한 `.cs` 파일.
2. **컴파일**: C# 컴파일러가 소스 코드를 **IL(중간 언어)**로 변환.
3. **실행**: CLR이 IL을 읽어 기계어로 번역하며 실행.

## 3. 환경 설정
- **Visual Studio**: Microsoft에서 제공하는 강력한 IDE입니다. [C# 및 .NET 데스크톱 개발] 워크로드를 설치해야 합니다.
- **SDK**: .NET 앱을 빌드하고 실행하는 데 필요한 도구 모음입니다.
