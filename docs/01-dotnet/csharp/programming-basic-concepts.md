# [1장] 프로그래밍 기본 개념

## 1. 프로그램 (Program)
컴퓨터가 수행할 **명령문의 집합**입니다. 실행 파일 형태로 존재하며 운영체제에 의해 순차적으로 실행됩니다.

## 2. 저수준 언어 vs 고수준 언어
- **저수준 언어**: 컴퓨터(CPU)가 직접 이해하는 기계어나 어셈블리어. 하드웨어 제어에는 강력하지만 사람이 작성하기 어렵습니다.
- **고수준 언어**: C#, Java, Python 등 사람이 이해하기 쉬운 언어. 생산성이 높고 유지보수가 쉽습니다.

## 3. 컴파일 (Compile)
고수준 언어로 작성된 소스 코드를 컴퓨터가 이해할 수 있는 언어(기계어 또는 중간 코드)로 번역하는 과정입니다.

## 4. 디버깅 (Debugging)
프로그램 코드 내에 존재하는 오류(Bug)를 찾아내고 수정하는 과정입니다.
