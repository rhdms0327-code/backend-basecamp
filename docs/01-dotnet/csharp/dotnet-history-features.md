# [.NET 분석] .NET의 역사와 주요 특징

.NET은 2002년부터 시작되어 현재까지 끊임없이 진화해 온 강력한 개발 플랫폼입니다.

## 1. .NET Framework (2002 ~ 2019)
- **시작**: Microsoft가 Java에 대항하여 출시한 Windows 전용 플랫폼입니다.
- **특징**: 폐쇄형 소스였으며, Windows 시스템에 깊게 통합되어 운영되었습니다.
- **현재**: 4.8.1 버전을 마지막으로 기능 추가보다는 유지보수와 안정성 중심으로 운영됩니다.

## 2. .NET Core (2016 ~ 2019)
- **등장 배경**: 클라우드 환경의 발전과 크로스 플랫폼(Linux, macOS) 지원 요구에 따라 탄생했습니다.
- **특징**: 오픈 소스로 전환되었으며, 모듈화된 설계를 통해 가볍고 빠른 성능을 제공합니다.

## 3. 통합된 .NET (2020 ~ 현재)
- **.NET 5/6/7/8**: .NET Framework와 .NET Core의 복잡한 명칭을 정리하고 하나로 통합한 플랫폼입니다.
- **최신 상태**: 현재는 **.NET 8 (LTS)** 버전이 가장 널리 권장되며, 성능과 생산성 면에서 가장 뛰어난 상태입니다.

## 4. .NET의 핵심 가치
- **오픈 소스**: 전 세계 개발자와 함께 발전합니다.
- **크로스 플랫폼**: 하나의 코드로 Windows, Linux, macOS, iOS, Android 앱을 개발할 수 있습니다.
- **고성능**: Kestrel 웹 서버 등 최신 기술을 통해 동급 최강의 성능을 보여줍니다.
