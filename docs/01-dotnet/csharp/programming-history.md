> 컴퓨터와 프로그래밍 언어가 발전해 온 흐름을 살펴봅니다.


## 1. 기계식 계산기 시대
- **파스칼 계산기**: 최초의 기계식 계산기 (톱니바퀴 방식).
- **해석기관 (찰스 배비지)**: 현대 컴퓨터 구조(입력-연산-제어-출력)의 개념을 처음으로 제시했습니다.

## 2. 소프트웨어의 근간
- **튜링 기계 (앨런 튜링)**: 알고리즘과 소프트웨어의 이론적 모델을 제시했습니다.
- **천공 카드**: 초기 데이터 입력 및 저장 수단으로 사용되었습니다.

## 3. 언어의 발전
- **기계어**: 0과 1로 된 저수준 언어.
- **어셈블리어**: 기계어를 사람이 읽기 쉬운 기호로 표현.
- **고수준 언어 (C, C++, C#, Java 등)**: 사람이 이해하기 쉬운 문법으로 작성하며, 컴파일 과정을 통해 실행됩니다.
