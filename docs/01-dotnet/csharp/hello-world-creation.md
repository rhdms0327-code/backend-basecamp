# [2장] Hello World 프로젝트 생성

## 1. 프로젝트 생성 순서
1. **Visual Studio 실행**: [새 프로젝트 만들기] 클릭.
2. **템플릿 선택**: [콘솔 앱] 선택.
3. **설정**: 프로젝트 이름과 저장 위치 지정.
   - 팁: [솔루션 및 프로젝트를 같은 디렉터리에 배치]를 체크하면 폴더 구조가 단순해집니다.
4. **프레임워크 선택**: .NET 8.0 (LTS) 권장.

## 2. 최상위 문 (Top-level statements)
- C# 9.0부터 도입된 기능으로, `Main` 메서드와 클래스 선언 없이 바로 코드를 작성할 수 있습니다.
- 학습 시에는 생략된 구조를 명확히 이해하기 위해 **[최상위 문 사용 안 함]**을 체크하여 직접 구조를 작성해 보는 것을 권장합니다.

## 3. 빌드 및 실행
- **F5**: 디버깅 모드로 실행.
- **Ctrl + F5**: 디버깅 없이 실행 (학습 시 권장).
