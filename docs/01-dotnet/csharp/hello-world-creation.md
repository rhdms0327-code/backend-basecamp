# 🎯 [2장] Hello World 프로젝트 생성

> C# 학습의 첫 걸음인 Hello World 프로젝트를 비주얼 스튜디오에서 생성하는 단계별 절차와 주요 옵션 설정 방법을 상세히 정리합니다.

## 1. 프로젝트 생성 순서
1. **Visual Studio 실행**: [새 프로젝트 만들기] 버튼을 클릭합니다.
2. **템플릿 선택**: 검색창에 '콘솔 앱'을 입력하고 C# 전용 템플릿을 선택합니다.
3. **설정**: 프로젝트 이름과 저장 위치를 지정합니다. 
   - 팁: '솔루션 및 프로젝트를 같은 디렉터리에 배치'를 체크하면 폴더 구조가 더 깔끔하게 생성됩니다.
4. **프레임워크 선택**: 가급적 장기 지원 버전인 **.NET 8.0 (LTS)**을 선택합니다.

## 2. 최상위 문 (Top-level statements)
- C# 9.0부터 도입된 기능으로, 복잡한 네임스페이스와 클래스 선언 없이 바로 로직을 작성할 수 있습니다.
- **학습 권장 사항**: 프로그램의 전체적인 뼈대를 정확히 파악하기 위해, 처음에는 **[최상위 문 사용 안 함]** 옵션을 체크하여 표준 구조를 직접 작성해 보는 것이 좋습니다.

## 3. 빌드 및 실행
- **F5**: 디버깅 모드로 실행합니다. 브레이크 포인트를 활용한 분석이 가능합니다.
- **Ctrl + F5**: 디버깅 없이 실행합니다. 학습용 코드를 빠르게 확인할 때 권장됩니다.

> [!TIP]
> 비주얼 스튜디오 하단의 '출력' 창에서 컴파일 과정의 로그를 확인할 수 있으며, 에러 발생 시 해당 내용을 클릭하여 즉시 코드 위치로 이동할 수 있습니다.
