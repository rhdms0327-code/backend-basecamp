> - **연산자 (Operator)**: 동작을 수행하도록 지시하는 기호나 키워드 (예: `+`, `is`, `new`).

- **식 (Expression)**: 하나 이상의 피연산자와 연산자의 조합으로, 최종적으로 **하나의 값을 반환**합니다.
- **문 (Statement)**: 어떤 동작이나 행위를 수행하는 코드의 단위입니다. (예: `if`, `for`, 선언문).

## 2. 람다 연산자 (=>)
- 매개변수 목록과 본문을 구분하는 역할을 합니다.
- **람다 식**: 값을 반환하는 익명 함수로 사용됩니다.
- **식 본문 멤버**: 메서드나 프로퍼티의 본문이 짧을 때 중괄호 대신 사용합니다.

## 3. 연산자 우선순위
- 산술 > 관계 > 논리 > 할당 순으로 우선순위가 결정됩니다.
- 괄호 `()`를 사용하여 명시적으로 우선순위를 조절하는 것이 가독성 측면에서 좋습니다.
