# [.NET 분석] .NET 용어 정리

.NET 생태계에서 자주 쓰이는 핵심 용어들을 정리했습니다.

## 1. 프레임워크 (Framework) vs 라이브러리 (Library)
- **프레임워크**: 애플리케이션의 기본 구조(아키텍처)를 제공합니다. 개발자는 그 구조 위에 기능을 얹어 개발합니다. (제어의 역전)
- **라이브러리**: 특정 기능을 수행하는 코드의 집합입니다. 개발자가 필요할 때 직접 호출하여 사용합니다.

## 2. 웹 애플리케이션 (Web Application)
- 웹 브라우저를 통해 사용하는 소프트웨어입니다.
- **구성 요소**: 클라이언트(브라우저), 웹 서버, 애플리케이션 서버, 데이터베이스.

## 3. .NET의 두 가지 의미
1. **전체 생태계**: Microsoft의 모든 개발 플랫폼을 통칭합니다.
2. **최신 플랫폼**: .NET Framework와 .NET Core를 통합한 최신 버전(.NET 5/6/7/8+)을 의미합니다.
