> Visual Studio에서 ASP.NET MVC 5 개발 환경을 구축하고, 첫 번째 웹 애플리케이션 프로젝트를 생성하는 단계를 안내합니다.

## 1. 개발 환경 구축

Visual Studio Installer를 통해 필요한 워크로드를 설치해야 합니다.

1. **워크로드**: 'ASP.NET 및 웹 개발' 선택.
2. **설치 세부 정보**: 우측 메뉴에서 **'.NET Framework 프로젝트 및 항목 템플릿'**을 반드시 체크해야 MVC 5 템플릿이 나타납니다.

## 2. 프로젝트 생성 설정

1. **템플릿**: 'ASP.NET 웹 애플리케이션 (.NET Framework)' 선택.
2. **프레임워크**: 학습을 위해 .NET Framework 4.5~4.8 중 하나를 선택합니다.
3. **구조**: '솔루션 및 프로젝트를 같은 디렉터리에 배치'를 체크하면 폴더 구조가 단순해집니다.

## 3. 웹 애플리케이션 구성

- **프레임워크**: **MVC**를 선택하고, 우측의 핵심 참조 추가에서도 'MVC'가 체크되었는지 확인합니다.
- **인증**: 학습 단계이므로 '없음'을 선택하여 자동 생성되는 복잡한 코드를 최소화합니다.
- **옵션**: 'HTTPS에 대한 구성'과 '단위 테스트 프로젝트 만들기'를 체크하는 것을 권장합니다.
